Etiquetas extras

Etiqueta pre

La etiqueta pre sirve para respetar los espacios y líneas en blanco, al contrario de la etiqueta párrafo

Hola Soy Anya Lol 

 Hola   Soy  
        Anya    Lol

Etiqueta abr

La etiqueta abr se utiliza para las abreviaturas

Según datos de la OMS el consumo excesivo de azúcar provoca daños severos en la salud de lo niños

Etiqueta bdo

Si se desea cambiar la dirección de un texto, se utiliza la etiqueta bdo

Este es un texto normal

Este es un texto diferente

Etiqueta cite

La etiqueta cite se utiliza para escribir citas textuales

Exraño a leon por: Anya Nuñez

DABBBB por: Anya Nuñez

Etiqueta small

La etiqueta small hace el texto más pequeño

Este texto está dentro de una etiqueta small

Etiqueta sub

Si se desea escribir un subíndice, se debe utilizar la etiqueta sub

LOL LOL

Etiqueta sup

Si se dea escribir un superíndice se debe utilizar la etiqueta sup

LOL LOL

Etiqueta time

Para escribir horas y fechas se debe utilizar la etiqueta time

Mañana es día 2024-11-06

Mañana es día 6 de noviembre de 2024

Etiqueta address

Para especificar al navegador que el texto se refiere a una dirección, se utiliza la etiqueta address

Loma dorada No.100 Loma de Rosales CP 89110 Tampico, Tams.
